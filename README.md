# IMDb TV Show Rating Heat Map Generator
This is a rating heat map generator for TV Shows based on IMDb ratings using webscraping in Python with Selenium.

## Run
`.\run [series_id]`

### Example
`.\run tt1024810`

You can get the series ID from the IMDb link
